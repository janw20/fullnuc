# FullNuc

CLI tool to output full nuclear PDFs from bound proton PDFs.

## Installation
`fullnuc` only depends on pandas. Install by cloning the git repo:
```sh
$ git clone https://github.com/janw20/fullnuc.git
```
and then, for example, symlinking `fullnuc.py` into `~/.local/bin`:
```sh
$ ln -s /path/to/fullnuc/fullnuc.py ~/.local/bin/fullnuc.py
```
and then show the CLI arguments:
```sh
$ fullnuc.py --help
```
